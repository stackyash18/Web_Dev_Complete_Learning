Client side scripting and server side scripting

Client Side Scripting
->executes on user's device.
->primarily javascript,HTML,CSS.
->makes clicks and scrolls works.
->can't access server directly.
->quicker for UI changes.


Server Side Scripting
->executes on a remote machine.
->PHP,Python,Java,Nodejs,etc.
->makes clicks and scrolls works.
->can't access server directly.
->quicker for UI changes.
